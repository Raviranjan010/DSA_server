## 🎯 Daily Study Routine

### Weekday Routine (2-3 hours):
```
Hour 1: Theory
  • Read notes (30 min)
  • Study diagrams and dry runs (15 min)
  • Understand code examples (15 min)

Hour 2-3: Practice
  • Solve 2-3 problems (60-90 min)
  • Analyze complexity (10 min)
  • Review mistakes (10 min)

Last 10 min: Plan tomorrow's topics
```

### Weekend Routine (4-5 hours):
```
Hour 1-2: Deep Practice
  • Solve 4-5 problems
  • Focus on weak areas

Hour 3: MCQ Test
  • Complete topic MCQs
  • Review explanations

Hour 4: Revision
  • Review week's notes
  • Update personal cheatsheets

Hour 5: Planning
  • Plan next week
  • Join coding communities
  • Watch supplementary videos
```
