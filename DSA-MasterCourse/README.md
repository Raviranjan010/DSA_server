## 📚 How to Use This Repo

### For Each Topic:
1. **Read the Notes** (`XX_notes.md`)
   - Follow the 13-section structure sequentially
   - Don't skip the dry run — trace manually
   - Understand the "why" before the "how"

2. **Test Your Understanding** (`XX_mcqs.md`)
   - Attempt all 30+ MCQs without looking at answers
   - Read explanations even for correct answers
   - Note down concepts you got wrong

3. **Study the Code** (`code/` folder)
   - Read brute force → optimized → most optimal
   - Type code from scratch (don't copy-paste)
   - Run test cases and verify output

4. **Solve Practice Problems** (listed in notes)
   - Start with Easy, progress to Hard
   - Spend 30 minutes per problem before checking solution
   - Solve each problem at least twice

5. **Quick Revision** (before interviews)
   - Use cheatsheets in `24_Quick_Revision/`
   - Review company-specific questions in `23_Interview_Questions_by_Company/`

### Problem-Solving Strategy:
```
Step 1: Read problem carefully (2 min)
Step 2: Identify the pattern/topic (3 min)
Step 3: Think of brute force approach (5 min)
Step 4: Optimize using patterns you learned (10 min)
Step 5: Write code with comments (10 min)
Step 6: Test with edge cases (5 min)
Step 7: Analyze time/space complexity (5 min)
```

## ⚠️ Prerequisites

### Required:
- **Basic programming knowledge** (variables, loops, functions)
- **High school mathematics** (basic algebra, logarithms)
- **A computer with C++ compiler** (GCC, Clang, or MSVC)
- **Willingness to practice daily** (consistency > intensity)

### Recommended (but not required):
- Basic understanding of functions and recursion
- Familiarity with any programming language (Python, Java, etc.)
- Knowledge of basic math (summation, series)

### Setup Instructions:
```bash
# Install C++ compiler (choose one):

# Option 1: GCC (Linux/Mac)
sudo apt-get install g++        # Linux
brew install gcc                # Mac

# Option 2: MinGW (Windows)
# Download from: https://www.mingw-w64.org/

# Option 3: MSVC (Windows)
# Download Visual Studio: https://visualstudio.microsoft.com/

# Compile and run:
g++ -std=c++17 filename.cpp -o output
./output                        # Linux/Mac
output.exe                      # Windows
```
