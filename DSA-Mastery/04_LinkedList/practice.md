**Mistake Categories**:
1. **Null Pointer**: Not checking if node exists
2. **Lost Reference**: Forgetting to save next pointer
3. **Wrong Return**: Returning wrong node as head
4. **Memory Leak**: Not deleting removed nodes
5. **Infinite Loop**: Cycle not handled properly
6. **Off-by-One**: Wrong number of iterations
7. **Edge Case**: Empty list, single node not handled

## ⏱️ Time Pressure Guidelines

### 45-Minute Interview Structure

**Minutes 0-10: Understand & Clarify**
- Read problem carefully
- Ask clarifying questions:
  - "Is it singly or doubly linked?"
  - "Can the list be empty?"
  - "Are there cycles possible?"
  - "Should I free deleted nodes?"
- Discuss brute force first
- Talk about optimization

**Minutes 10-30: Code**
- **DRAW THE LIST FIRST!**
- Write clean, readable code
- Use meaningful variable names (prev, current, next)
- Comment pointer changes
- Handle basic edge cases

**Minutes 30-40: Test**
- Test with 3-4 examples
- Include edge cases (empty, single node)
- Walk through pointer changes step by step
- Fix bugs

**Minutes 40-45: Optimize & Analyze**
- Discuss time/space complexity
- Suggest optimizations
- Mention alternative approaches
